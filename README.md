# OpenGL Project - Eid ul Fitr Celebration Scenario


This project is a Computer Graphics application developed using **C++ and OpenGL**.  
It represents a festive **Eid ul Fitr celebration scene** with animated graphical elements, designed to demonstrate core computer graphics concepts such as rendering, animation, and scene composition.




## ✨ Features

🕌 Mosque scene design  
🎇 Fireworks animation  
🌅 Day-Night cycle  
👨‍👩‍👧 Celebrating people and children  
🌍 Real-time graphical rendering using OpenGL

## 🛠️ Tech Stack

💻 C++  
🟢 OpenGL  
🔷 GLUT


## 💻 How to Run
1. Open project in Code::Blocks 
2. Install OpenGL (GLUT required)
3. Compile and run `src/main.cpp`
